Text Color As Per Color Tag

<red>I'm inside red tags</red>
Result: I'm inside red tags 
<green>I'm inside green tags</green>
Result: I'm inside green tags 
<aqua>I'm inside aqua tags</aqua>
Result: I'm inside aqua tags

(Use any color tag , it will work for all)

<p color="blue">I'm having color attribute as blue</p>
Result: I'm having color attribute as blue
